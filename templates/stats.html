4. Індекси читабельності та сприйняття складності

Індекс читабельності Флеша (Flesch Reading Ease) — це узагальнений показник, що оцінює, наскільки легко текст сприймається під час читання: чим вищий бал, тим простішим є текст для розуміння. Шкала вимірюється у діапазоні від 0 до 100.

Індекс ґрунтується на двох структурних характеристиках мови: середній довжині речення (слов у реченні) та середній кількості складів у слові. Для української мови застосовується адаптована форма класичної формули Флеша:

R = 206.835 − 1.3 × (слів / речення) − 60.1 × (складів / слово)

Коефіцієнти 1.3 та 60.1 підібрані для української мови, оскільки її фонетико-морфологічні особливості (зокрема, більша середня кількість складів у слові, активне словотвірне суфіксування, інша «середня» довжина речення) систематично зсувають розподіли метрик порівняно з англійською. Для інших мов використовують відмінні коефіцієнти, щоби зберегти узгодженість шкали (0–100) із реальним відчуттям складності носіїв мови.

Нижче наведено інтерпретаційні діапазони індексу. Вони дають практичне читання значенням: до якого умовного «освітнього рівня» та виду тексту ближче наведений матеріал.

Окрім класичних показників читабельності, для більш повної оцінки складності тексту використано інтегральний індекс сприйняття (Perception Index). Цей показник відображає не лише формальну структуру речень, а й лексичну щільність — тобто співвідношення змістових та службових слів, що безпосередньо впливає на когнітивне навантаження під час читання.

На відміну від традиційних індексів читабельності, які враховують лише довжину речень та складність слів, індекс сприйняття інтегрує два ключові аспекти:

  • структурну читабельність (плавність синтаксису, довжину речень),
  • лексичну насиченість (частку змістових слів у тексті).

У сучасній обчислювальній лінгвістиці подібні комбіновані моделі широко використовуються для оцінки когнітивної складності текстів та прогнозування того, наскільки легко чи важко читачу сприймати інформацію.

Для поточного аналізу застосовано вагову модель, у якій:

  • 60% становить структурна читабельність тексту,
  • 40% — лексична щільність (частка змістових слів).

Такий розподіл ваг є збалансованим: він враховує як синтаксичну побудову, так і інформаційну насиченість. Для офіційних промов, які часто поєднують складні речення зі значною концентрацією термінів, ця модель забезпечує більш реалістичну оцінку складності, ніж будь-який із показників окремо.

Формально індекс визначається так:

P = 0.6 × R + 0.4 × C

де R — читабельність у діапазоні 0–1 (нормалізоване значення адаптованого індексу Флеша), C — частка змістових слів (content words ratio).
Після обчислення результат знову масштабується до інтервалу 0..100.

Узагальнюючи результати статистичного аналізу, можна стверджувати, що текст має {% if stats.readability.ua_index < 40 %} високу складність і потребує підготовленого читача. {% elif stats.readability.ua_index < 60 %} помірну складність, що є типовим для офіційних чи аналітичних виступів. {% else %} високу доступність і добре підходить для широкої аудиторії. {% endif %}

Лексичний профіль {% if stats.vocabulary.lexical_diversity_ratio > 0.6 %} демонструє багатий і різноманітний словник, {% else %} характеризується переважанням стійких формулювань, властивих офіційним текстам, {% endif %} а частотний аналіз підтверджує ключові теми промови.
